## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in `react-vite-starter-kit`, please report it to us as described below.

### Reporting Process

Please report security vulnerabilities via [GitHub Security Advisories](https://github.com/klabacher/react-vite-starter-kit/security/advisories/new).

If that is not possible, please open a [GitHub Issue](https://github.com/klabacher/react-vite-starter-kit/issues/new) with the label `security` and do not include sensitive details in the public issue. We will contact you for more information.

### Response Time

We are committed to responding to security reports within **48 hours**.

### Disclosure Policy

We request that you do not publicly disclose the vulnerability until we have had a chance to verify and fix it. We will work with you to coordinate the public disclosure.
